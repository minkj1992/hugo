# Hugo 모듈

- Hugo 모듈은 Go모듈로 구동됩니다.

## `Go`모듈

go-moduels는 vgo, dep, glide 등으로 파생되던 버전 관리의 표준을 제시하였으며, gopath를 더는 사용하지 않을 수 있도록 해주는 도구입니다. **즉 패키지간 종속성 관리를 golang team이 표준화하여 관리합니다.**
